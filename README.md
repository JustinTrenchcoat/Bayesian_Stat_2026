# Bayesian Statistics Class Project
This is the GitHub page for Bayesian Statistics Spring 2026 Class Project.

Inspired by [my past project](https://github.com/JustinTrenchcoat/MATH441_Project_2), this project focuses on modelling future car price  by utilizing Bayesian Statistics techniques.

This Github Page contains code, visualizations, presentation slides and a full report in NeurlIPS format.
